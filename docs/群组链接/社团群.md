## 南京理工大学社团/同好会群聊信息

>[!TIP]
>别社团/组织/同好会由于历史问题已被注销或从未注册  
>但往往具有较多成员、有组织架构、会定期组织活动，具有一定影响力，因此此处不做区分一并列出。    
>具体社团名单以团委清单为准。
>如信息有缺漏，请自行提交 PR 或在评论区指出。

**团委领导下的南京理工大学的社团管理绝对是南理工一大特色，不得不品。**

原则上讲，加入社团是受团委系统控制的，每人最多加三个社团。    
不过大部分社团的普通成员不需要走团委手续，一般认为加群即入社。
